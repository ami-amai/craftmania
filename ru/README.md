## Введение

* Игра действует по [Основным правилам](#основные-правила) с [Набором правил](#набор-правил)

* Игроки накапливают [Ресурсы](#предмет) и создают из них [Предметы](#предмет), чтобы получить преимущество

* Каждый ход игрок может использовать несколько доступных [Действий](#действие) для нужных ему вещей

* Установка [Предметов](#предмет) на [Поле](#поле) происходит с помощью [ХП](#хп)

## Терминология

### Правила

#### Основные правила
* Правила, которые обычно работают с любым Набором правил и являются неизменными, и описывают основные механики игры

#### Набор правил
* Правила, содержащие [Предметы](#предмет) в игре, пути их Создания и Получения, паттерны и стоимости Атак, действия любых иных [Предметов](#предмет) и может быть изменено

### Игра

#### Поле
* Игровое поле, на котором происходят все основные действия в игре
* Размер задается [Набором правил](#набор-правил)

#### Инвентарь
* Игровой инвентарь, до 9 слотов, в который можно ложить [Предметы](#предмет) с [Поля](#поле)
* Размер задается [Набором правил](#набор-правил)

#### Действие
* Активность, которую можно выполнить во время хода
* [Действия](#действия) задаются [Основными правилами](#основные-правила)

[**Действия**](#действия)

#### Предмет
* Любой игровой предмет, участвующий в крафтах либо дающий какие-либо возможности в игре
* [Предметы](#предмет) указаны в [Наборе правил](#набор-правил)
* Дополнительный функционал в рамках [Основных правил](#основные-правила) может быть прописан в [Наборах правил](#набор-правил)
    * Ресурс - предмет, расходуемый во время [Создания](#создание)
    * Атакующий предмет - предмет, позволяющий производить атаку

#### ХП
* Блок здоровья (Health Point Block / HPb (HP) / HpB (HB)), необходимый для установки [Предметов](#предмет) на [Поле](#поле), а также может расходоваться для каких-либо [Действий](#действие)

## Действия

### Размещение

* [Действие](#действие), направленное на изменения положения [Предметов](#предмет) в игре

#### Установка

* Размещение [ХП](#хп) на [Поле](#поле)
    * Пустой [ХП](#хп)
    * С [Предметом](#предмет)

#### Перемещение

* Движение [Предметов](#предмет) по [Полю](#поле)
    * Инвентарь -> [Поле](#поле) (пустой [ХП](#хп))
    * [Поле](#поле) -> Инвентарь (пустой слот)

#### Перестановка

* Перестановка [Предметов](#предмет) между друг другом
    * [Поле](#поле) <-> Инвентарь
    * [Поле](#поле) <-> [Поле](#поле)

#### Атака

* [Действие](#действие) производится любым [Размещением](#размещение) атакующего [Предмета](#предмет) на [Поле](#поле), расходует [ХП](#хп) и разрушает определенным патерном все [ХП](#хп), указанным в [Наборе правил](#набор-правил)
* Для атаки можно расходовать свои [ХП](#хп) с [Поля](#поле), но все [Предметы](#предмет) на этих [ХП](#хп) будут разрушены
* Атака может иметь собственные способы и условия разрушения в соответствии с текущим [Набором правил](#набор-правил)

### Получение

* [Действие](#действие), которым можно получить [Ресурс](#предмет)
    * В инвентарь (пустой слот)
    * На [Поле](#поле) (пустой [ХП](#хп))

### Создание

* [Действие](#действие), с помощью которого можно создавать новые [Предметы](#предмет) из [Ресурсов](#предмет)
    * Все [Ресурсы](#предмет) должны находиться на [Поле](#поле) рядом друг с другом (в радиусе 1 блока)
    * Создание некоторых [Предметов](#предмет) требует наличия других [Предметов](#предмет) на [Поле](#поле)
    * Все [Ресурсы](#предмет) после [Создания](#создание) разрушаются

### Разрушение

* [Действие](#действие), уничтожающее [Предметы](#предмет)
    * Бесплатно, взамен ничего
    * За [Действие](#действие), взамен 1 [ХП](#хп)
